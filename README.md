# Admin Nexus

This is a web application built with Vite, React, TypeScript, and Shadcn UI.

**Live Demo:** [admin.ahmedlitim.com](https://admin.ahmedlitim.com)

## Prerequisites

- [Node.js](https://nodejs.org/en/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or any other package manager

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ahmed-litim/admin-nexus.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

This will create a `dist` folder with the production-ready files.
